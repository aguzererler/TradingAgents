# Active Plans

This directory contains active implementation plans. Each plan documents a sequence of steps for a multi-step task.

## How to Use

- Plans are markdown files named descriptively (e.g., `pipeline-implementation.md`).
- Each plan has a status: `active`, `completed`, or `paused`.
- Before starting work, check if an active plan exists for your task.
- Do not skip steps unless explicitly instructed by the user.
